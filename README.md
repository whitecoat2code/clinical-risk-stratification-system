# Clinical Risk Stratification System

## 📌 Overview
This project simulates a basic clinical decision system that classifies patient risk levels using vital parameters like BP and cholesterol.

## ⚙️ Features
- Generates synthetic patient dataset
- Classifies patients into Low, Medium, High risk
- Flags critical patients
- Provides basic statistical insights

## 🛠️ Tech Used
- Python
- Pandas
- NumPy

## 📊 Output
- Risk distribution
- Average BP of high-risk patients
- Age and cholesterol insights
